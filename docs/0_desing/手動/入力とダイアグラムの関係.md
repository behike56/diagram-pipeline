# 入力とダイアグラムの関係

## 入力ごとにどんな図を作ると良いか

* **CloudFormation / Terraform**

  * メイン: コンポーネント図 / デプロイメント図
  * 例: VPC / Subnet / ALB / ECS / RDS / S3 / IAM Role などの関係
* **Kubernetes マニフェスト**

  * メイン: コンポーネント図
  * 例: Ingress → Service → Deployment → Pod → DB の関係
* **アプリケーションソースコード**

  * メイン: クラス図 / コンポーネント図 / シーケンス図
  * 例: Controller → Service → Repository → 外部API / DB
